# Rapport — [TD3]

## Points clés

### 1. [Modèle événement, état, rendu]

J'ai compris qu'avec Javascript, il y a un enchainement d'événements pour produire une interaction. D'abord un événement (clic sur un bouton), puis il y a une modification d'état (ouvert ou fermé) enfin, le navigateur met à jour l'événement, en le montrant visuellement. 
Exemple : Bouton "Afficher plus". 
Evenement : Clique sur le bouton.
Etat : Variable qui change true ou false.
Rendu : JavaScript ajoute ou retire une classe.

### 2. [ClassList.toggle]

Cette méthode est plus simple pour créer une interaction. Sans toucher au JavaScript, on peut créer une interaction comme un menu burger.
